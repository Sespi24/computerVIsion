Project #2: IMAGE MANIPULATION

Before moving onto computer vision, we need to first have the ability to manipulate images. In this project, I used the OS library to allow the user to select an "images" folder. Then, OpenCV and NumPy manipulated the colors of images through swapping matrix values and masking. Finally, Matplotlib allowed the user to see the manipulated image (although I could have used OpenCV for the same purpose). Below is an example image and three manipulated versions:

The first image is the normal red-green-blue (RGB) image
The second was switched from an RGB to a hue-saturation-value (HSV) color space
The third is a mask of the first image, capturing most of the person's skin tone
The fourth is a negative version of the image (all values are reversed)
