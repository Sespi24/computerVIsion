Project #0 (?): VSCODE AND GITHUB

For me to begin my programming journey in Python, I needed to get to know my tools first. GitHub is a programming standard, and I chose to use VisualStudios Code for my IDE because of how well it fits with Windows systems. While this is not a "project" of sorts I would be remissed if I did not at least mention it here at the start because of the lessons it taught. 1) Seemingly random errors are very frustrating. Sometimes the best way to approach an issue is take a step back and not try the same thing over and over. And 2) turning towards others for help is crucial. Asking questions, prodding, and actually reading tutorials instead of copying and pasting code snippets is huge. As the saying goes, I am standing on the shoulders of giants, so through some debugging and assistance, I was able to tackle programming milestones.

Project #1: APIS AND DATA VISUALIZATION

The first big project was communicating with and pulling information from a NBA API, utilizing NumPy to give some summary statistics, and visualize some of the data with MatPlotLib. Shown to the right is an image of the two graphs made from the NBA data.

Project #2: IMAGE MANIPULATION

For computer vision, we need to first have the ability to manipulate images. In this project, I used the OS library to allow the user to select an "images" folder. Then, OpenCV and NumPy manipulated the colors of images. Finally, MatPlotLib allowed the user to see the manipulated image (although I could have used OpenCV for the same purpose). To the left is an example of the manipulated image.
